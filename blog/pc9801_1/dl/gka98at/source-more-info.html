This is a PC-98 to PS/2 keyboard converter based on a PIC16f84(a) or 16f648a microcontroller. I found this stuff on:
http://shinku.ddo.jp/wordpress/?page_id=4151.

That page is worth a visit if you can get through the language barrier because it is a solid write-up and features a patch for a larger PIC model (16f648).

The original site was down, and the asm file from that site was not archived by the wayback machine. However, the webpage the file was hosted on was archived (wayback machine link).

Hasu's TMK keyboard firmware documentation for its own PC98 to USB keyboard converter implementation references the original site, which is how I discovered the converter existed to begin with.

I have not tried to build this converter, but I expect it works. I'd like to build one since I prefer PS/2 over USB keyboards anyway. But for now I have made a TMK converter which works.

As of this writing, you can find PIC16F84(A)'s from:

  • Mouser:
    • 10MHz (PIC16F84)
    • 20MHz (PIC16F84A)
  • Digi-Key:
    • 10MHz (PIC16F84)
    • 20MHz (PIC16F84A)

You can find PIC16F648A's from:

  • Mouser:
    • 20MHz
  • Digi-Key:
    • 20MHz

They (either kind of microcontroller) must be rated at least 10MHz. I think from the PDF maybe these are actually getting clocked at 6.144MHz via crystal? - But that might be changable to 10MHz in the source code. I did not look closely at all.
